CodeCompass - Carmack

(Version: Carmack, Revision: )

CodeCompass is a tool that helps you understand large scale software projects by providing many navigation, visualization, query possibilities in the source code.

What's new, what's good?

Highlights

  • Full Java support
  • Full Regex Search
  • Log Search
  • Recursive Function Pointer Analysis
  • Searchable & Navigable GIT revision history
(Click on the image)

Search

  • Regular expression in text searches.
    -Regex can be written between / / signs, like /.*ipsec.*/
  • Log message search: Support for searching the place where a log message (possibly) generated.
    -Just copy paste a log message in the search bar
  • Searching for C++ qualified names in definition search.
    -You can search for names such as “charging::process”
  • Filtering by programming language.
    -Filter for languages in searches like Python, C++
  • Filtering by definition type (function, type, etc.)
  • Faster search

New Language Support

  • Java: show variable read/write, show usages of types, show function calls/callers
    -Ask for info tree after right clicking a java symbol
  • Delos: type analysis, process diagrams
    -Right click in a Delos file and ask for usage of a symbol
    -Process diagrams:Right click on an MI and ask for process diagram
  • CDF/MI Diagrams
    -Neighbors of an MI
    -Users of an MI
    -Mis used by an MI
    -Process/DBN diagram in an MI

New C++ Functions

  • Function pointer based call chain exploration
    -Ask for info tree for a function and you can recursively ask for caller and called functions
  • Module dependency diagrams
    -Right click on a directory in the “Project Tree” and see the available diagrams
    -Internal Architecture of a module
    -Other modules a module depends on
    -Users of a module
  • Basic template support
    -Show instantiations of templates

Revision Control

  • Searchable & Navigable GIT revision history
    -Search among GIT commits
    -Blame view: Show who changed a file line-by-line

GUI

  • Yet more ergonomic design
