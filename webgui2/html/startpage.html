CodeCompass - Earhart

(Version: Earhart, Revision: )

CodeCompass is a tool that helps you understand large scale software projects by providing many navigation, visualization, query possibilities in the source code.

What's new, what's good?

Highlights of the Earhart Release

  • Code Metrics McCabe complexity, lines of code, number of codechecker bugs
  • C++ Class Catalogue Shows all classes in your project organized by namespaces
  • Pointer Alias Detection Shows which pointers and references point to a variable
  • UML Class overview diagramShows the UML class relationships between classes defined under a directory

New C/C++ Features

  • Pointer Aliases Click on a variable and ask for "Pointer Aliases Diagram". This diagram shows which pointers and references point to a given variable. Using this feature you can track where your variable may be modified. See also the aliases subtree in the info tree.
  • UML Class overview diagram If you click on a file or directory in the File Manager, you can ask for "UML class overview diagram", which shows the inheritance and containment relationship between the classes defined in the selected directory.
  • Additional target support Codecompass supports new gcc compilationt targets: powerpc, mips.
  • Class Collaboration Diagram Class collaboration diagram for a class shows recursively the related classes and their inheritance and containmnet relationships.

Generic Features

  • Code Metrics Spot your most problematic modules by asking for "Metrics" for a file or directory. A treemap view will show the the distribution of codechecker faults or McCabe complexity organized by directory.
